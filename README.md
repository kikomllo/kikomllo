### - Featured Projects:

#### 🎮 [Role Playing Engine (RPE) - (WIP)](https://github.com/kikomllo/RPE/tree/main)
A modular 2D Game Engine built in **Java**. I developed this to explore the secrets of **decoupled architecture**. It features a custom Game Loop and a data-driven system that loads game objects via JSON.
* *Key Focus:* OOP Patterns, Component-Based Design, Systems Architecture.

#### ⚛️ [Molecular Particle Simulation](https://github.com/kikomllo/particleSimulation)
A high-performance **C++/OpenGL** simulation modeling atomic interactions. I implemented the **Lennard-Jones potential** to understand how to translate physical molecular forces into efficient, real-time code.
* *Key Focus:* Physics Integration, Memory Management, OpenGL Shaders.

#### 🧊 [WebGL 3DEngine](https://github.com/kikomllo/3dEngine)
Built out of a fascination with **3D-to-2D projection**. This engine uses pure **JavaScript and WebGL** to render models. I wrote a custom OBJ parser and implemented the linear algebra for transformation matrices manually.
* *Key Focus:* Linear Algebra, Rendering Pipeline, Graphics Programming.

#### 🛰️ [OGameTool Interface Enhancer](https://github.com/kikomllo/OGame-Tool)
A **JavaScript** UserScript focused on browser automation. It injects custom UI elements and automates fleet calculations by interacting directly with the game's DOM.
* *Key Focus:* Asynchronous JS, DOM Manipulation, UX Enhancement.
