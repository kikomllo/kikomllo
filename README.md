# Hi, I'm Francisco Melo 👋

A Computer Science student driven by a deep curiosity for how software works under the hood.

### My Engineering Philosophy 
Most of my projects are born from a "first-principles" approach:

* **Graphics & Projection:** My interest in **3D-to-2D projection** led me to build a WebGL engine from scratch to understand transformation matrices and the graphics pipeline.
* **Scientific Simulation:** I developed a **Molecular Particle Simulator** in C++ to explore how physics equations like the Lennard-Jones potential translate into high-performance code.
* **Systems Architecture:** I transitioned **CL-Core** from a monolithic script into an event-driven MQTT network to master decoupling, microservices, and asynchronous design.

### Featured Projects

#### [Voice and Proximity Framework (CL-Core) - (WIP)](https://github.com/kikomllo/CL-Core)
An asynchronous voice control and BLE presence automation system built in Python. I initially developed this to master BLE signals and proximity tracking, and it organically evolved into a decoupled MQTT microservice network. It features local STT inference (Whisper), a custom NLP intent engine, and isolated edge nodes that react seamlessly to both voice commands and room proximity.
* *Key Focus:* Python Asyncio, MQTT Pub/Sub, NLP Intent Routing, BLE Radar Automation.

#### [Molecular Particle Simulation](https://github.com/kikomllo/particleSimulation)
A high-performance **C++/OpenGL** simulation modeling atomic interactions. I implemented the **Lennard-Jones potential** to understand how to translate physical molecular forces into efficient, real-time code.
* *Key Focus:* Physics Integration, Memory Management, OpenGL Shaders.

#### [WebGL 3D Engine](https://github.com/kikomllo/3dEngine)
Built out of a fascination with **3D-to-2D projection**. This engine uses pure **JavaScript and WebGL** to render models. I wrote a custom OBJ parser and implemented the linear algebra for transformation matrices manually.
* *Key Focus:* Linear Algebra, Rendering Pipeline, Graphics Programming.

#### [OGameTool Interface Enhancer](https://github.com/kikomllo/OGame-Tool)
A **JavaScript** UserScript focused on browser automation. It injects custom UI elements and automates fleet calculations by interacting directly with the game's DOM.
* *Key Focus:* Asynchronous JS, DOM Manipulation, UX Enhancement.
